Display remote site posts in the Elementor page builder

== Description ==

The "Remote Site Posts for Elementor" plugin gives an Elementor widget to display remote site posts in the page. You should provide remote site URL, no. of posts and search value. It can fetch data from any remote WordPress site by the use of the WordPress Rest API.

Features:
*   Control to no. of Posts.
*   Display posts specific to search value.
*   Easy to use Elementor Widget.

Elementor is most popular page builder plugin of WordPress. This plugin developed to enhance the capability of the Elementor plugin.

== Installation ==

1. Upload `remote-site-posts-for-elementor` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress Admin dashboard
1. Edit any post type with the "Edit with Elementor" button.
1. Add new widget "Remote Site Posts" from the Basic widget group
1. Fill The URL of a remote WordPress site.
1. Please fill other settings as per your requirement
1. Hit the `Publish` button


== Frequently Asked Questions ==

= Should the remote site build on WordPress? =

Yes, The Remote site should build upon WordPress.

= Will it works if remote site disabled? =

No, It won't work If the remote site has disabled the REST API.

= Is it legal to display other sites posts? =

Yes.
